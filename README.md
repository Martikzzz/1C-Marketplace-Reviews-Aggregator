# 1C-Marketplace-Reviews-Aggregator

Данный проект представляет собой программное расширение для типовой конфигурации «1С:Управление торговлей 11.5». Модуль предназначен для автоматизации сбора, обработки и визуализации клиентских отзывов и рейтингов с внешних маркетплейсов (Ozon, Wildberries) непосредственно в учетной системе.

📌 Основные функции

Интеграция с API: Автоматический сбор данных через HTTP-запросы (протокол TLS 1.2).

Парсинг JSON: Десериализация входящих данных во внутренние структуры 1С с использованием объекта ЧтениеJSON.

Защита от дублей: Интеллектуальный механизм проверки уникальности отзывов по ВнешнийID.

Визуализация (звезды): Отображение графического рейтинга и цветовая индикация отзывов («Единое окно» в карточке номенклатуры).

Аналитическая отчетность: Построение диаграмм на базе СКД для оценки качества ассортимента.

Автоматизация: Возможность фонового обновления данных с помощью регламентных заданий.

🛠 Технологический стек

Платформа: 1С:Предприятие 8.3.27+

Базовая конфигурация: Управление торговлей, ред. 11.5

СУБД: PostgreSQL (рекомендуемая), файловый режим (тестовый)

Формат обмена: JSON

Архитектура: Расширение конфигурации (.cfe)

📂 Структура репозитория

/src — Исходный код расширения, выгруженный в файлы (XML/BSL).

/release — Готовый файл расширения Дип_АгрегацияОтзывов.cfe.

/docs — Техническое задание и пояснительная записка (фрагменты).

🚀 Установка

Загрузите файл .cfe из папки /release.

В Конфигураторе 1С перейдите в меню Конфигурация -> Расширения конфигурации.

Нажмите Добавить, укажите имя и снимите флажки «Безопасный режим» и «Защита от опасных действий».

Нажмите Конфигурация -> Загрузить конфигурацию из файла и выберите скачанный .cfe
